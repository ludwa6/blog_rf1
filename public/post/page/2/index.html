Deleting Content

July 09, 2019
TechHow
The “Blogdown” package to RStudio provides a nice easy UI (i.e. its “Addins”" menu) to push content thru Hugo to the web (via Github and Netlify)… But it has no affordance for the deletion of posts; for this you must go into the Hugo filesystem… Not so easy as i had hoped, alas, but doable. Here’s how: In RStudio’s file browswer, navigate to /content/post/ , tick box of content item(s) you want gone, and click Delete.
